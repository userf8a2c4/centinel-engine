## Grant targets

CENTINEL is suitable for:
- **Gitcoin / Octant**: civic-tech, open-source, democracy infrastructure
- **Open Society Foundations**: election integrity, Latin America programs
- **NDI / IRI**: electoral monitoring technology
- **NSF / OAS**: open-source research infrastructure
- **Bug bounties**: security audit programs (Immunefi, HackerOne)
